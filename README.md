# SplitShot

Local-first desktop app for competition shooting video analysis, merge, scoring, and export.

## Status

This repository contains:

- The implementation plan derived from `Directions.md`
- A Python 3.12 / PySide6 application
- Feature-focused automated tests

## Quick Start

```bash
uv sync
uv run splitshot
```

## Test

```bash
uv run pytest
```
